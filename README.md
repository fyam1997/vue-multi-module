### Usage

```shell
pnpm add @fyam/ui-utils
pnpm add @fyam/code-utils
```

### Setup

```shell
pnpm i
```

### Develop

```shell
pnpm run dev
```

### Publish

```shell
pnpm login
```

```shell
pnpm run build
pnpm publish --access public --filter code-utils
pnpm publish --access public --filter ui-utils
pnpm run delpoy-gh-pages
```
